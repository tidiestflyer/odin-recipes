## html
This repo's purpose is for me to practice.
From what I've seen, HTML really isn't that complicated, as long as you have a dictionary of the different tags, you can hobble a site together
All you have to remember are a few key things:
- Some users may use Screen Readers
- Use HTML semantics
- Always include a boilerplate
- Try to acheive the best search engine rating possible
